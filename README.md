# Style Editor

## Development

### Install dependencies
```bash
npm i
```

Add your mapbox access token to `ACCESS_TOKEN` in `.env.local`

```bash
cp example.env.local .env.local
```

### Commit
To commit use [git-cz](https://github.com/streamich/git-cz)

```bash
git-cz
```

## Deploy
```bash
npm run deploy
```
